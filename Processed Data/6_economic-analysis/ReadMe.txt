Excel sheet for calculating annual mileage, TOC, GHG and air pollutant emissions, and externalities for each emission from Daily Vehicle Mile Traveled for vehicles by type and fuel.

The coefficients are referenced to the "Afleet" spreadsheet provided by the Algonne Institute. 
url: https://greet.es.anl.gov/afleet

The file for the calculation is "[Format]Cost Calculation_ver3.xlsx".

The files beginning with "[Result]" are the actual calculation results for each scenario obtained from the optimization model.
